##### Function: `>=`
Return true if the value on the left is greater than or equal to the value on the right

Examples:

  3 >= 2
  -- Result: true

  1 >= 1
  -- Result: true

  'a' >= 'b'
  -- Result: false

###### Signatures
    a, a -> checkbox
