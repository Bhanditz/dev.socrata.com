##### Function: `random_number_normal`
```
  Returns a random float using a normal distribution with the mean and variance supplied: random_number_normal(mean, variance)

Examples:


  random_number_normal(0, 10)
  -- Result: "2.12030422"



```

###### Signatures
    number, number -> number
