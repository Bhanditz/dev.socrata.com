##### Function: `#NOT_BETWEEN`
```
  Return true if the left is not within the range of the right values

Examples:

  2 NOT BETWEEN 0 and 1
  -- Result: true

  1 NOT BETWEEN 0 and 2
  -- Result: false

  'b' NOT BETWEEN 'a' and 'c'
  -- Result: false




```

###### Signatures
    a, a, a -> checkbox
