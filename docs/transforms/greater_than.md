##### Function: `>`
```
  Return true if the value on the left is greater than the value on the right

Examples:

  3 > 2
  -- Result: true

  1 > 1
  -- Result: false

  'a' > 'b'
  -- Result: false




```

###### Signatures
    a, a -> checkbox
