##### Function: `upper`
```
  uppercase a string

Examples:

  upper('something')
  -- Result: "SOMETHING"

  upper('SOMETHINg')
  -- Result: "SOMETHING"

  upper(text_column)
  -- Result: "TEXT COLUMN DATUM"




```

###### Signatures
    text -> text
