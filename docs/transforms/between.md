##### Function: `#BETWEEN`
Return true if the left is within the range of the right values

  Examples

    1 BETWEEN 0 and 2 -- true
    2 BETWEEN 0 and 1 -- false
    'b' BETWEEN 'a' and 'c' -- true

###### Signatures
    a, a, a -> checkbox
