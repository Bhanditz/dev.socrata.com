##### Function: `<=`
```
  Return true if the value on the left is less than or equal to the value on the right

Examples:

  1 <= 2
  -- Result: true

  1 <= 1
  -- Result: true

  'a' <= 'b'
  -- Result: true




```

###### Signatures
    a, a -> checkbox
