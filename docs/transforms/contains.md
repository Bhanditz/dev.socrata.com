##### Function: `contains`
tell whether or not a string contains another string

Examples:

  contains('foobar', 'foo')
  -- Result: true

  contains('foobar', 'bar')
  -- Result: true

  contains('foobar', 'baz')
  -- Result: false

###### Signatures
    text, text -> checkbox
