##### Function: `replace`
```
  replace text with another piece of text

Examples:

  replace('some-text-literal', '-', '_')
  -- Result: "some_text_literal"

  replace(text_column, 'datum', 'data')
  -- Result: "text column data"




```

###### Signatures
    text, text, text -> text
