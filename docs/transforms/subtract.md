##### Function: `-`
```
  Subtract a number from another


  Change a number's sign

Examples:

  4 - 2
  -- Result: "2"

  0 - 2
  -- Result: "-2"


  -(-2)
  -- Result: "2"

  -(2)
  -- Result: "-2"




```

###### Signatures
    number, number -> number
    number -> number
